# MoviesReviews

MoviesReviews — веб-приложение на Django для публикации отзывов о фильмах. 
Пользователи могут регистрироваться, добавлять фильмы, оставлять и редактировать отзывы,
управлять профилем. Адаптивный интерфейс на Bootstrap 5.

## 📌 Возможности

- Просмотр списка фильмов с пагинацией.
- Детальная страница фильма с постерами и всеми отзывами.
- Регистрация и вход в личный кабинет.
- Добавление нового фильма (только для авторизованных пользователей).
- Написание, редактирование и удаление своего отзыва (один отзыв на фильм от одного пользователя).
- Профиль пользователя: аватар, биография, смена пароля.
- Админ-панель Django для управления всем контентом.
- Адаптивный дизайн на Bootstrap 5 (корректно отображается на телефонах и планшетах).

## 🛠 Требования

- Python 3.10 или новее
- pip
- Рекомендуется использовать виртуальное окружение (venv)

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий или скачайте архив проекта.**
   ```bash
   git clone https://github.com/Karasyantia/MoviesReviews.git
   cd MoviesReviews

2. **Создайте и активируйте виртуальное окружение.**
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # macOS/Linux:
   source venv/bin/activate

3. **Установите зависимости.**
   ```bash
   pip install -r requirements.txt

4. **Примените миграции базы данных.**
   ```bash
   python manage.py migrate

5. **Создайте суперпользователя (администратора).**
   ```bash
   python manage.py createsuperuser

6. **Запустите сервер разработки.**
   ```bash
   python manage.py runserver

7. **Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/**

## 🗂 Краткий обзор страниц
| Страница                    | URL                               | Доступ                |
|-----------------------------|-----------------------------------|-----------------------|
| Главная (список фильмов)    | `/`                               | Все                   |
| Детальная страница фильма   | `/movie/<id>/`                    | Все                   |
| Добавить фильм              | `/movie/add/`                     | Только авторизованные |
| Оставить отзыв              | `/movie/<id>/review/add/`         | Только авторизованные |
| Редактировать/Удалить отзыв | `/review/<id>/edit/` и `/delete/` | Только автор отзыва   |
| Регистрация                 | `/accounts/register/`             | Гости                 |
| Вход                        | `/accounts/login/`                | Гости                 |
| Выход                       | `/accounts/logout/`               | Авторизованные (POST) |
| Личный кабинет              | `/accounts/profile/`              | Авторизованные        |
| Мои отзывы                  | `/accounts/profile/my-reviews/`   | Авторизованные        |
| Редактирование профиля      | `/accounts/profile/edit/`         | Авторизованные        |
| Смена пароля                | `/accounts/profile/password/`     | Авторизованные        |
| Админ-панель                | `/admin/`                         | Авторизованные        |

## 🎨 Дизайн и адаптивность
Сайт построен на HTML-шаблонах с CSS-фреймворком Bootstrap 5.
Благодаря этому интерфейс автоматически подстраивается под экраны смартфонов, планшетов и настольных компьютеров.

## 🧩 Дополнительная информация
- Все формы имеют серверную и клиентскую валидацию.
- Пароли хранятся в зашифрованном виде (используется встроенная система аутентификации Django).
- Загружаемые изображения сохраняются в папке `media/`.
- Проект использует Class-Based Views и следует принципу DRY.
